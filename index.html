VisPilot

A common operation during this exploration is drill-down.
However, drill-downs could lead to inductive fallacies.
We present VisPilot, an accelerated visual data exploration tool.
It will guide analysts through the key insights in a dataset, while avoiding drill-down fallacies.
Get Started Live Demo Documentation

Features

Interesting Visualizations

VisPilot helps analysts discover interesting visualizations, understand attribute importance, and predict unseen visualizations better than other multidimensional data analysis baselines.

Clear Design

There are only a few Buttons! Specify the dataset name, x-axis value, y-axis value and the number of visualizations you want, we'll do the rest for you.

Fast Operation

It might take a while only when you run a dataset for the first time. After that, if you run on the same dataset or expand on the current one, the dashboard will be generated immediately.

Advanced Features

[Comming soon] You will be able to upload your own dataset and explore it using VisPilot!

Example

  • Insightful Dataset Summary

    After selecting the desired attributes to visualize and requesting for k number of visualization summaries, VisPilot automatically generates a dashboard consisting of a set of visualizations conveying the key insights in the dataset.

  • Iterative Drill-Down Refinement via Node Expansion

    Users can request for additional summaries to be generated based on the selected visualizations.

  • Signalling Visualization Subpopulation Size During Exploration

    While exploring the dashboard, it is useful to get a sense of the subpopulation size, i.e. the number of rows used for creating the aggregate visualization. VisPilot displays the subpopulation size as the border color of each visualization node (darker blue means larger subpopulation size). Users can also inspect the exact subpopulation size by hovering over the visualization node.
